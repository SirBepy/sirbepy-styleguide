# Styleguide Adoption Prompt

> This file is maintained by the developer. Edit it to describe your project's design system and styleguide conventions for Claude to adopt.

Please review the project's `assets/styles/styleguide.scss` and ensure the following:

1. All new components follow the token and spacing conventions defined in the styleguide.
2. Color, typography, and spacing variables from `styleguide.scss` are used consistently throughout the codebase.
3. Flag any hardcoded values that should be replaced with styleguide tokens.
4. Leave a brief summary of what was reviewed and any changes made.
